# บทที่ 1 Introduction
##  บทนำ
ในยุคที่ข้อมูลถูกสร้างขึ้นอย่างรวดเร็วจากทุกกิจกรรมของมนุษย์ ไม่ว่าจะเป็นการทำธุรกรรมออนไลน์ การใช้งานเครือข่ายสังคม การทำงานของอุปกรณ์ IoT ไปจนถึงงานวิจัยทางวิทยาศาสตร์ ปริมาณข้อมูลได้เพิ่มขึ้นจากระดับ เทราไบต์สู่เพตะไบต์ ทำให้การวิเคราะห์ข้อมูลด้วยวิธีการเดิมไม่เพียงพออีกต่อไป แม้เราจะมีข้อมูลจำนวนมาก แต่ “ข้อมูลจำนวนมากไม่ได้มีความหมาย หากเราไม่สามารถค้นพบความรู้จากมันได้”

ตรงนี้เองที่ Data Mining เข้ามามีบทบาทสำคัญ
Data Mining คือกระบวนการค้นหารูปแบบ ความสัมพันธ์ หรือความรู้ที่ซ่อนอยู่ภายในข้อมูลขนาดใหญ่ โดยอาศัยเทคนิคจากหลายสาขา เช่น Machine Learning, สถิติ, ระบบฐานข้อมูล และการประมวลผลประสิทธิภาพสูง เพื่อให้ได้มาซึ่งข้อมูลเชิงลึกที่นำไปใช้ประโยชน์จริงได้

บทนี้จะปูพื้นฐานให้เข้าใจ

ทำไม Data Mining จึงมีความจำเป็น

กระบวนการสกัดความรู้ (KDD Process)

ประเภทของข้อมูลและรูปแบบของ pattern ที่ค้นหาได้

ฟังก์ชันสำคัญของการทำเหมืองข้อมูล

ตัวอย่างประยุกต์ใช้งานในโลกจริง

และประเด็นท้าทายที่ยังต้องแก้ไขในสาขานี้

เนื้อหาทั้งหมดจะช่วยให้ผู้อ่านเห็นภาพรวมของ Data Mining ตั้งแต่เหตุผลที่ต้องมี เทคนิคที่ใช้ ไปจนถึงความสามารถและขอบเขตของมัน ซึ่งเป็นพื้นฐานสำคัญก่อนจะเข้าสู่บทถัดไปของวิชานี้

##   Data Mining คืออะไร?
Data Mining คือกระบวนการค้นหารูปแบบ ความรู้ หรือข้อมูลเชิงลึกจากข้อมูลจำนวนมาก  
Pattern ที่ดีต้องมีลักษณะ:
- ไม่ trivial
- ซ่อนอยู่ในข้อมูล
- ไม่เคยรู้มาก่อน
- ใช้ประโยชน์ได้จริง

---

##   ทำไมต้องมี Data Mining?
- ข้อมูลเพิ่มขึ้นแบบก้าวกระโดด (TB → PB)
- ธุรกิจ วิทยาศาสตร์ และสังคมสร้างข้อมูลมหาศาล
- มนุษย์วิเคราะห์เองไม่ไหว  
-> จำเป็นต้องใช้เครื่องมืออัตโนมัติ

---

##   กระบวนการ KDD
กระบวนการ KDD ย่อมาจาก Knowledge Discovery in Databases (การค้นพบองค์ความรู้ในฐานข้อมูล) มันคือขั้นตอนที่เป็นระบบเพื่อดึงเอา รูปแบบ (patterns) ที่มีประโยชน์และ องค์ความรู้ (knowledge) ออกมาจากข้อมูลดิบจำนวนมหาศาล

---

##   ประเภทของข้อมูลที่นำมาทำเหมืองข้อมูล
- ฐานข้อมูลแบบตาราง
- Data warehouse
- Time-series, sequence
- Text, Web
- Multimedia
- Graph & Social networks
- Sensor / Streaming data

---

##  ฟังก์ชันหลักของ Data Mining
- Generalization  
- Pattern discovery (association rules)  
- Classification  
- Clustering  
- Outlier detection  
- Sequential / trend analysis  
- Graph & network mining  

---


##  ประเด็นสำคัญใน Data Mining
- ข้อมูล noise / missing  
- ความหลากหลายของข้อมูล  
- ความเป็นส่วนตัว (Privacy)  
- Algorithm ต้อง scale ได้  
- Visualization ทำยากเมื่อข้อมูลหลายมิติ  

---


###  Data Exploration vs Data Mining
| หัวข้อ | Exploration | Mining |
|--------|------------|--------|
| จุดประสงค์ | ดูภาพรวม | หา pattern |
| ความลึก | ตื้น | ลึก |
| เทคนิค | OLAP | ML algorithms |

---

###  Descriptive vs Predictive
| ประเภท | ความหมาย | ตัวอย่าง |
|--------|----------|----------|
| Descriptive | อธิบายสิ่งที่เกิดขึ้น | Clustering |
| Predictive | ทำนายอนาคต | Classification |
